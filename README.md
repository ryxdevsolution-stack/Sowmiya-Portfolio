## ğŸš¦ Getting Started

### Prerequisites
- Node.js 18.19.1 or higher (Note: Latest Next.js recommends Node 20.9.0+)
- npm or yarn

### Installation

1. Clone the repository:
```bash
cd Sowmiya-Portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```
